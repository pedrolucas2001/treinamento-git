Quais os comandos para configurar usuário e email no git?

Para usuario: git config --global user.name "Pedro"

Para email: git config --global user.email "pedro.lukas321@gmail.com"

Qual a diferença entre git e github?

Git é o aplicativo de versionamento

GitHib é onde fica armazenado seus repositorios para quem quiser acessar remotamente

O que faz o comando commit?

Envia pro repositorio remoto a alteração feita
